Hey JS Developers!

LuvvieScript is a small but seriously interesting early stage project that will challenge your preconceptions about you should structure large Javascript applcations and give you deep insight into different ways to think about Javascript apps. Jump right in and see how we are transpiling Erlang to Javascript.

Its open source so join us. Don't worry you don't need to be an Erlang guru.

Erlang In The Browser

LuvvieScript is a compile-to-javascript dialect of Erlang that runs in a browser. It is desgined to deliver the simplicity of Erlang's message passing and pattern-matching paradigm to the developers of large scale web systems - a front-end language that seamless works with high-availability Erlang server clusters for fast and powerful web development in a DevOps/Continuous Deployment friendly manner.
